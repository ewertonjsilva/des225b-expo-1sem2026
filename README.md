Um projeto educacional desenvolvido em React Native com Expo para a disciplina DES225B - Desenvolvimento de Sistemas. Este repositório contém atividades práticas, exemplos e exercícios realizados durante o 1º semestre de 2026.

## 📋 Descrição

Este projeto serve como material de apoio para o aprendizado de React Native utilizando o framework Expo. Inclui uma série de atividades e exemplos que demonstram conceitos fundamentais do desenvolvimento mobile cross-platform.

## 🚀 Funcionalidades

### Atividades (atividades/)
- **Atividade 01**: Introdução ao React Native com Expo
- **Atividade 02**: Componentes e Estilização Básica
- **Atividade 03**: Navegação e Layouts
- **Atividade 04**: Estados e Eventos
- **Atividade 05**: Formulários e Entrada de Dados
- **Atividade 06**: Listas e Scroll
- **Atividade 07**: Componentes Avançados

### Exemplos (exemplos/)
- Exemplos práticos complementando as atividades
- Demonstrações de componentes e funcionalidades

### Revisão (revisao/)
- Exercícios de revisão e consolidação de conceitos

## 🛠️ Tecnologias Utilizadas

- **React Native**: Framework para desenvolvimento mobile
- **Expo**: Plataforma para desenvolvimento React Native
- **JavaScript**: Linguagem de programação
- **React**: Biblioteca para construção de interfaces

## 📋 Pré-requisitos

Antes de começar, você vai precisar ter instalado em sua máquina:

- [Node.js](https://nodejs.org/en/) (versão 18 ou superior)
- [Expo CLI](https://docs.expo.dev/get-started/installation/)
- Um dispositivo móvel ou emulador para testar o app

## 🔧 Instalação

1. Clone este repositório:
```bash
git clone https://github.com/seu-usuario/des225b-expo-1sem2026.git
```

2. Entre no diretório do projeto:
```bash
cd des225b-expo-1sem2026
```

3. Instale as dependências:
```bash
npm install
```

4. Inicie o servidor de desenvolvimento:
```bash
npm start
```

## 📱 Como Usar

Após iniciar o servidor, você pode:

- **Web**: Pressione `w` no terminal para abrir no navegador
- **Android**: Pressione `a` para abrir no emulador Android
- **iOS**: Pressione `i` para abrir no simulador iOS (apenas macOS)

Use o aplicativo Expo Go no seu dispositivo móvel para escanear o QR code e testar o app.
